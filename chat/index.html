LAMP Database .ORG - Chat

Is on FreeNode!

HowTo

  • Install an IRC Client or use FreeNode WebIRC.
  • Join the Channel #lamp-database.org with the following command:
    /join #lamp-database.org
  • Switch to your desired nickname:
    /nick <YourNick>
  • Register your IRC nick:
    /msg NickServ REGISTER <YourPassword> <youremail@example.com>
